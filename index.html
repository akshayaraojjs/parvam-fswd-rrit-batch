There are 3 types of CSS:

  • Internal CSS
  • External CSS
  • Inline CSS

Internal CSS: It is written within HTML document using style tag

External CSS: It is written in the separate CSS file

Inline CSS: It is written within the HTML tags using style attribute

We have Selectors in CSS for applying styles to the target element

We can use:

  • Element Selector - Just use element directly
  • Class Selector - Use (.)class-name
  • ID Selector - Use (#)id-name
  • Universal Selector - Use (*)
